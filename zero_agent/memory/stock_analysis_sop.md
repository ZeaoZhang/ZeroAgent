# 股票分析技能 SOP

## 概述
本 SOP 指导 Agent 使用 ZeroAgent 的股票分析技能（Stock Analysis Skill）对 A 股、港股、美股进行技术分析和行情查询。

## 可用工具

| 工具名 | 功能 | 适用场景 |
|--------|------|----------|
| `analyze_stock` | 全面的单只股票技术分析 | 用户问"分析XX股票"、"XX怎么样" |
| `get_stock_quote` | 获取实时行情报价 | 用户问"XX现在多少钱"、"XX行情" |
| `analyze_trend` | 快速趋势检查（轻量版） | 用户问"XX趋势如何"、"XX走势" |

## 使用流程

### 场景一：用户请求分析单只股票
1. 调用 `analyze_stock(code="600519")` 获取完整分析
2. 从返回结果中提取：信号、趋势状态、量能状态、操作建议
3. 用自然语言向用户呈现结果

### 场景二：用户请求比较多只股票
1. 对每只股票分别调用 `analyze_stock`
2. 汇总结果进行比较，指出强弱差异

### 场景三：用户请求行情数据
1. 调用 `get_stock_quote(code="AAPL")`
2. 展示价格、涨跌幅、最高最低价

## 股票代码格式
- A股: 纯数字，如 `600519`（贵州茅台）、`000001`（平安银行）
- 港股: 5位数字，如 `00700`（腾讯）
- 美股: 字母代码，如 `AAPL`（苹果）、`TSLA`（特斯拉）

## 注意事项
- 分析前先确认用户提供了股票代码
- 如果用户只说了股票名称（如"茅台"），先尝试推断代码或询问用户
- 分析结果包含置信度，低置信度时建议用户结合其他信息判断
- 本技能技术分析仅供参考，不构成投资建议
